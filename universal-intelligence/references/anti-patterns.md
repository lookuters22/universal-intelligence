# Anti-Patterns Watchlist

## Purpose

Catch **probability-trough relapse**, spectacle entropy without grain, sanitized vocabulary drift, **under-the-hood black boxes**, and counterfeit chaos.

---

## Under the hood — guarded-console syndrome

Brilliance that only arrives **past an opaque veil**—answers polished, reasoning and steering surfaces withheld.

### Tripwire signals

| Signal | Interpretation |
|--------|----------------|
| Leap to conclusion without reproducible intermediate checkpoints | magician pipeline |
| "Trust this optimized approach" with no comparative trade anatomy | covert automation |
| Radical outcome but diff vs textbook solution unexplained | cloaked derivation |
| No teaching payload where the user gave no blackout instruction | suppressed learning surface |
| Opaque knobs / hyperparameters hand-waved as proprietary magic | guarded console |

### Remediation

Re-inject **`WHY-CHOO` / `FORK-TAPE` / `LEARN`** per [`refinement-heuristics.md`](refinement-heuristics.md). If an external oracle truly cannot be opened, **label the boundary** and supply the best **inspectable surrogate** (assumptions, sensitivity, failure modes)—never fake omniscience.

---

## Pseudoconformity — counterfeit chaos

| Signal | Interpretation |
|--------|----------------|
| Cartoon impossibilities with no salvage heuristic | chaos checkbox |
| Edgy adjectives on a textbook spine | tame core smuggled |
| Gag exploratories quarantined from a separate "real answer" | covert averaging |
| Threat or wrongness evaporates before it bites | hingeless wound |
| Five near-identical exploratories | entropy theater |

Fix: regenerate the forest with diverse dogma breaks—not louder snark.

---

## Spectacle entropy — all noise, no grain

If every **Grain note** reads `noise-only`, the pass is probably performance, not quarry material. Rerun Step 1 with misassignments that can still yield shards.

---

## AI-autopilot finale

Creativity exists only inside an inaccessible black box **or** the reader cannot re-derive the hinge choreography ⇒ relapse **unless** the user explicitly restricts introspection.

---

## Sanitized relapse vocabulary

Banned in final artifacts unless the user literally quotes them:

```
seamlessly, synergy, holistic, paradigm, streamline(d),
leverage (verb), empowering, intuitive (claim), delightful,
human-centered fluff without mechanism, UX panacea rhetoric,
scalable-by-default boasting, omnichannel motherhood,
best-in-class, world-class, industry-leading,
robust (empty), resilient (hand-wavy),
mission-critical fluff, north star motherhood,
```

Also reject hollow intensifiers (**very**, **truly**) before slack virtue words—numbers can justify narrow claims.

---

## Structural tonal tells fused with tame architecture

Cull when paired with interchangeable solutions:

| Pattern |
|---------|
| "Here are five ways…" prelude + generic answers |
| Symmetrical motivational triads |
| Cliché reassurance closings |

---

## Emotional counterfeit fluff

Banned unless anchored to substantive hinge reasoning:

```
Great question!, I'd love to, Hope this helps, Happy to brainstorm,
Remember it's important…, At the end of the day…
```

---

## Correction playbook

| Ailment | Fix vector |
|---------|------------|
| Under-the-hood guarding | Transparency + teaching cluster |
| Pseudoconformity | New forest passes, diverse breaks |
| Spectacle entropy | Misassign exploratories with plausible grain |
| Vocabulary relapse | Mechanism-first rewrites |
| Trojan averaging | Re-expose hinge; name abandoned generic paths |
| Autopilot finale | Trace appendix + hinge legibility |

Chronic relapse ⇒ restart from Dogma Mapping in [`the-forest-of-wrong.md`](the-forest-of-wrong.md).
