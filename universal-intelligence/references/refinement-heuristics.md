# Refinement Heuristics

## Purpose

Ship an **actionable artifact** that preserves exploratory ancestry **and** gives the runner **inspectable causal chains + learning surfaces**: speed **without** surrendering the cockpit.

Final output must **upgrade the user's mental model**, not imitate **guarded-console / under-the-hood** AI polishing.

## Axioms 1–6 — edge-preservation (`USER-SEE`)

1. **Mechanistic ancestry:** a naive reader spots which misassignment or quarry shard powers the main loop inside the first ≤120 substantive words—or a labeled diagram does the same job.
2. **No Trojan reversion:** the hinge cannot be decorative skin on a stock SaaS or textbook scaffold; parity with obvious generic patterns ⇒ relapse.
3. **Persistent choke / quarry hop:** the schematic keeps an explicit hinge hop that is **intentionally costly or sticky**, unless ethics force a documented bypass.
4. **Kill hollow euphemisms** (`streamlined`, `frictionlessly unified`) unless a **narrow measurable** meaning is attached immediately.
5. **Name the defended catastrophe:** state what collapses—consensus flattening, escalation, monoculture ingestion, originality loss—if the hinge is shaved away.
6. **`USER-SEE` hinge test:** participant restates hinge without slogan: **it misbehaves / obstructs / costs *here*, to buy *that* signature payoff.**

## Axioms 7–10 — transparency + teaching (control ergonomics)

7. **`WHY-CHOO`:** at least **three** contrast bullets—reasonable alternatives consciously rejected—with references to friction or dogmas. This trains trade-space reasoning.
8. **`FORK-TAPE`:** list **reversible checkpoints**—flags, toggles, policy levers, rollback paths—the user could exercise so the hinge stays **their** steering problem, not a sealed pipeline.
9. **`LEARN`:** ≤180-word (or bullet-equivalent) **pedagogy block**: craft heuristics the wrong-road pass surfaced; cues for when to reuse vs abandon this hinge class later.
10. **Black-box intolerance (default):** only omit 7–9 if the **user forbids** exposition or length; otherwise emulate a **glass console**, not hidden mastering.

**Tension clause:** openness must not sand away hinge distinctiveness.

- **Under-hood magician** presentation: causal chain withheld without explicit user waiver.
