# Friction Analysis

## Purpose — engineer review stance

Operate like an experienced practitioner on a **post-exploratory review sweep** after the naive / wrong-road dump finished:

- Inputs are predominantly **mess / wrong-road material** deliberately—do not morally rescue yet.
- Task: map **Pain Coordinates** *and* note **localized fascinating wrong fragments**—micro motifs that snag attention *because* orthodoxy would veto them—with **zero synthesized fixes**.
- Disciplined survey + diagnosis: flag *oddly compelling shards* alongside honest catalogue of breakage.

Forbidden during this phase
